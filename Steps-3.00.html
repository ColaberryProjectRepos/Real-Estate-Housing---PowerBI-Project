Opening Jupyter notebook and importing necessary libraries

  • Open Jupyter notebook instance of python3 and browse for Housing Project.ipynb file from the extracted files
  • This ipynb file is opened in web browser and looks like the picture shown below

 


  • The following libraries need to be imported in Jupyter notebook to continue with the project
    • requests
    • json
    • re
    • pandas 
    • numpy 
    • plotly
    • pyodbc
    • sqlite3
    • sys
    • datetime
    • flatten_json
    • random
    • timeit
    • random
    • itertools
    • bs4

 

  • If any of the above library is not existing, we install the library using pip command (!pip install (library name))
    • Now the environment is set to carry out web scraping on Zillow, Century21, redfin and on realtor.
