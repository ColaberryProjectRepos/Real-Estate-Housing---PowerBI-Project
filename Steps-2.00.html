Steps to install SQL server and create a Database

  • Install SQL Server developer version https://www.microsoft.com/en-gb/sql-server/sql-server-downloads
  • Install SSMS from https://docs.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-2017

  • After opening SSMS connect to local instance of SQL Server 




  •  After Connecting to local SQL server, right click on the Database folder and create new database “Housing”

 

  


  • Now Housing database is created



  • Open the following SQL scripts using SSMS and execute the scripts. This will create the tables required in Housing Database to store the data mined from different websites.



  • Every year Section 8 publishes all the Zip Codes they cover in the DFW.  The zip codes are collected and stored in Zip Code Prices table in Housing database as shown below. By executing the ZipCodePrices.sql will create and  insert the zip codes and prices data into the table
